# **App Name**: VerdantCards

## Core Features:

- Deck Creation: Create new flashcard decks with a user-defined title.
- Flashcard Addition: Add individual flashcards to decks, including content for both the front and back sides.
- Study Mode: Review flashcards in a deck, marking each card as known or unknown for spaced repetition.
- Flashcard Editing: Edit existing flashcards to correct mistakes or improve clarity.
- Deck Organization: Allow users to categorize and organize their flashcard decks into custom collections for efficient study.

## Style Guidelines:

- Background color: Light Green (#E8F5E9) to create a calming and inviting study environment.
- Primary color: Forest Green (#43A047) for interactive elements and accents.
- Accent color: Yellow-Green (#A5D6A7) for secondary interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Use clean, simple icons to represent study functions.
- Maintain a clean, clutter-free layout with clear visual hierarchy.
